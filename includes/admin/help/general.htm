General Theme Set Up

Home Page

This theme uses a homepage template for the home page, in the options you will see a "Home Page Categories" option where you can select what categories you want to display on the home page. You can select multiple categories by holding the Ctrl key while selecting.

To setup your blog page all you have to do is add a new page and from the "Page Attributes" meta box, under "Template", select the "Blog" template and then click update.

If you don't want to use the homepage template as your homepage, you can switch to any other page from the WordPress Reading Settings. Go to "Settings" => "Reading Settings" and you will see the section where it says "A static page"

Header Logo

To use your own header logo go to "Appearance" => "Header" and you can upload your custom logo or you can click "Remove Header Image" and use your site title and tagline in general settings.

Background

To use your own backgound go to "Appearance" => "Background" and you can upload your custom background and choose a color or you can click "Remove Background Image" and use just a color of your choosing.

Menus

Leaf supports 3 menus, to make a custom menu go to "Appearance" => "Menus" and you can create a custom menu for your site. The "Primary Menu" is located just below the header of the site and uses SuperFish menu effects for the drop down submenus. The "Header Menu" is located just above the header and the "Footer Menu" is located just above the site copyright in the footer, both header and footer menus do not support drop down menus.

Widgets

To add widgets go to "Appearance" => "Widgets" and you can select the widgets you want to display in the sidebar.

The Main Sidebar widget area appears on posts and pages but not on the homepage when the homepage widget area has widgets.

The Home Page widget area appears on the homepage page, you can leave this empty to use the main sidebar on the home page.

The Header widget area appears on the right of your header logo, This is a good place to add a search widget, Social widget, or a header ad's(Banner 468 x 60) using the text widget.

The three footer widget area's appear in the sites footer.

Theme Options

The theme options are a great way to change the look and feel of your site. Most options are self-explanatory so im only going to run through a couple.

The "Theme Accent Color" option will change various colors on your site, mostly links and background colors. Warning: light & dark colors will blend in with icons, backgrounds, and text.

The "Sidebar Column" option will change the width of your sidebar. Four column is about 300px wide and three column is about 220px wide.

The "Custom CSS" option is one of the most important options in the theme for customizing your site. Code entered into the textarea is included right before the closing </HEAD> tag on each page of your site. The most important use for this area is to enter custom CSS rules to control the look of your site. Help with CSS
